# bbousuite: Easily Install and Load \`bbou\` (Boreal cariBOU) Packages

The \`bbou\` suite of packages are designed to allow analysis
(\`bbouretro\` and \`bboutools\`) and simulation (\`bbousims\`) of
Boreal Caribou survival and recruitment data ('bboudata') to estimate
population growth rates. A shiny app is also available (\`bboushiny\`).
